# 审慎断事框架

## 从结构到行动

1. 复述问题中的决策、期限和问卦者可控变量。
2. 读下卦为内在基础或早期条件，读上卦为外部环境或后期展开；这是一种默认映射，若情境不合应明确调整。
3. 用动爻位置识别变化发生在哪个层级，不凭单一关键词断吉凶。
4. 比较本卦与之卦：哪些性质增强、减弱或换位。
5. 用互卦提出一个“隐藏机制”假设；用错卦做反证；用综卦做换位思考。三者只作辅助，不压过本卦与之卦。
6. 把每个判断转成现实可验证的问题或动作。

## 行动导向的“天纪式”取向

本 Skill 仅把以下抽象原则作为独立设计取向，不归为任何人的逐字观点：重视自然变化、时位、验证、君子自省和现实行动；把卦视为观察关系与选择的工具，而非宿命判词。

推荐句式：

- “卦象结构提示你优先检查……，因为……”
- “如果现实中出现 A，可采取 B；若出现 C，则暂停并复盘。”
- “这是由‘下卦 X → 上卦 Y、某爻发动’推得的情境假设，不是事实认定。”

避免句式：

- “一定会 / 注定 / 百分之百”。
- 冒充古籍原文、课程原话或特定老师口吻。
- 用象义给人贴人格、疾病、犯罪或道德标签。
- 以再次起卦覆盖不喜欢的结果。

## 质量检查

- 排卦是否可由原始输入复算？
- 是否清楚区分事实、象义和推断？
- 是否至少给出一个可验证条件和一个止损/复盘点？
- 是否尊重用户自主决策？
- 是否对高风险事项设置了专业求助边界？
